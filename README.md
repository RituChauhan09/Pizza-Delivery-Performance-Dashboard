#### 📊 Delivery Performance \& Efficiency Dashboard (Power BI)

##### Overview

This is one of my first dashboards created while learning Power BI. The project focuses on analyzing delivery performance and efficiency. It helps in understanding delivery time, delays, and the impact of factors like traffic on overall operations.

The aim was to track key metrics and identify areas where the delivery process can be improved.

##### What I Built

* Designed an interactive dashboard to monitor delivery performance
* Tracked key metrics such as total orders, average delivery duration, and delayed orders %
* Compared delivery performance across different traffic conditions
* Added visualizations like charts and KPI cards for better insights

##### Key Insights

* High traffic leads to longer delivery time
* Low traffic results in faster deliveries
* Some orders are delayed, indicating scope for improvement
* Delivery efficiency varies based on external conditions like traffic

##### Tools Used

* Power BI
* Excel dataset
